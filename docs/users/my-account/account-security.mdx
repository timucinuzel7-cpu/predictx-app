## Security checklist

- Enable <strong>Two-Factor Auth</strong> in <strong>/settings/two-factor</strong>.
- Verify the site URL before signing messages.
- Never share seed phrase, private key, or backup codes.
- Revoke unused wallet connections periodically.

## Signature safety

Before approving any signature or transaction:

- verify network/account
- review destination/action
- confirm amount/token details

If anything looks unexpected, reject and contact support.
