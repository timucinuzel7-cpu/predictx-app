## Supported locales

Current supported set: `en`, `de`, `es`, `pt`, `fr`, `zh`.

English (`en`) remains enabled as default.

---

## What this section controls

| Control | Purpose |
| --- | --- |
| **Locale toggles** | Enables/disables non-default locales in your fork. |
| **Automatic translations** | Enables automatic translation for event titles and categories. |

---

## Important dependency

<Callout type="warning">
  Automatic translations require OpenRouter credentials configured in **Admin -> General**.
</Callout>

If OpenRouter is not configured, the automatic translations toggle stays unavailable.

---

## Recommended rollout

1. Enable only locales your team can moderate/support.
2. Configure OpenRouter in **Admin -> General**.
3. Enable automatic translations if needed.
4. Validate translated events/tags in each enabled locale.
