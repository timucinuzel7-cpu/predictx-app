## What this section controls

| Feature | Purpose |
| --- | --- |
| **Visibility toggle** | Hide/show events in public event lists. |
| **Livestream action** | Add, edit, or remove livestream URL per event. |
| **Sports final action** | Set sports final state and score for supported sports events. |
| **Sync settings** | Choose whether new synced events auto-deploy or require manual activation. |
| **Table filters** | Filter by main category, creator, and series. |

---

## Recommended operations flow

1. Review new events and confirm visibility status.
2. Set livestream links for live coverage events.
3. For sports moneyline events, update final status promptly.
4. Keep sync mode aligned with your moderation capacity.
