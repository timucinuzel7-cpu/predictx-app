title: Market Context

<Callout type="info">
  This section controls the generator itself. OpenRouter API key/model configuration lives in **Admin -> General**.
</Callout>

## What this section controls

| Control | Purpose |
| --- | --- |
| **Enable market context** | Turns generation on/off globally. |
| **Prompt template** | Defines instructions sent to the model. |
| **Variable insertion** | Injects live event/market placeholders into the prompt. |

---

## Setup flow

1. Configure OpenRouter API key/model in **Admin -> General**.
2. Open **Admin -> Market Context** and enable the feature.
3. Edit prompt template and include relevant variables.
4. Save and validate output on a real event.

---

## Model guidance

- Models with live web browsing usually generate stronger context for fast-moving markets.
- If using free OpenRouter endpoints, review privacy settings and provider policies before production use.
- Keep prompts short, explicit, and outcome-neutral.
