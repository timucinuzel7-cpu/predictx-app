## What this section controls

| Control | Effect |
| --- | --- |
| **Main Category** | Promotes a tag to main-category placement on the site. |
| **Hide Category** | Hides the category itself from category surfaces. |
| **Hide Events** | Hides events that contain this category tag. |
| **Translations** | Opens non-English label editor for category names. |

---

## Daily operations

1. Use search and sorting to find high-impact categories.
2. Keep only strong discovery buckets marked as **Main Category**.
3. Use **Hide Events** for emergency de-listing without deleting source data.
4. Maintain translations for enabled locales to keep discovery consistent.

---

## Notes

- English label remains the source value.
- Translation dialog stores localized names per non-default locale.
