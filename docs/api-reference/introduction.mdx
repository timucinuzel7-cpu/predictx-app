## Working with playgrounds

1. Open an endpoint page and fill in required parameters.
2. Click `Send request` to execute against the configured host.
3. Inspect response headers/body and copy the generated request for your client.

## Glossary

| Term | Definition |
| --- | --- |
| **Token / Asset ID** | Exposure to a specific market outcome. Winning outcome tokens redeem for 1 USDC at resolution. |
| **Market** | One prediction market defined by condition ID, token IDs, and metadata. |
| **Event** | Group of related markets under one theme. |
| **Slug** | Human-readable identifier used in URLs. |
| **CLOB** | Central limit order book used for matching and pricing. |

## Need help?

If you need unpublished routes or run into integration issues, contact the engineering team and include the request path, headers (without secrets), and response payload.
